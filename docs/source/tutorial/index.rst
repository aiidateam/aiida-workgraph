
Tutorials
===========================================

Explore real-world examples of building complex, reproducible computational workflows with `AiiDA-WorkGraph`.

* :doc:`/tutorial/autogen/materials_science_ase`
   Example workflows using `AiiDA-WorkGraph` and the `Atomistic Simulation Environment (ASE) <https://wiki.fysik.dtu.dk/ase/>`_.

   - :ref:`Atomization energy <ase_atomization_energy>`: A simple, linear workflow to calculate the atomization energy of a diatomic molecule.
   - :ref:`Equation of state (EOS) <ase_eos>`:  A more advanced workflow that demonstrates how to handle loops, dynamic inputs and outputs to calculate the equation of state for a bulk structure.
